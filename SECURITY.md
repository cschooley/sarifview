# Security Policy

## Reporting a Vulnerability

If you find a security issue in sarifview, please report it privately rather
than opening a public issue:

- GitHub private vulnerability reporting: https://github.com/cschooley/sarifview/security/advisories/new
- Email: chris@schooley.name

Please include steps to reproduce and the potential impact. We'll acknowledge
reports within a few days.

## Supported Versions

sarifview ships a single rolling release. Only the latest version published
to the VS Code Marketplace / Open VSX Registry is supported.
